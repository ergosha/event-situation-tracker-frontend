Frontend application for displaying events and situation state from the
Event Situation Tracker backend.

## Tech Stack
----------
- Next.js (App Router)
- React
- TypeScript
- Fetch API

## Features
--------
- Fetch and display events from backend
- Client-side rendering using Next.js Client Components
- Type-safe API consumption with TypeScript

## Running Locally
---------------
Prerequisites:
- Node.js 18+

Step 1: Install dependencies
npm install

Step 2: Start development server
npm run dev

Frontend will be available at:
http://localhost:3000

## Backend Dependency
------------------
This frontend expects the backend to be running at:
http://localhost:8080

Make sure the backend is started before opening the frontend.

## Project Structure
-----------------
- app/: Next.js App Router pages

## Notes
-----
- The frontend is intentionally kept simple
- Styling and advanced state management are out of scope for this version
- The backend repository is separate and can be run independently
